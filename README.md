Echo des Oubliées — Podcast Website

Echo des Oubliées is a web platform designed to showcase podcasts that amplify women’s voices and share stories of resilience, culture, and society. This project features a responsive layout, audio playback, search functionality, and categorized podcast listings.

Project Overview

The website presents podcasts in a structured and visually appealing way, allowing users to:

Browse top podcasts and categories.

Play audio episodes directly on the page.

Search for episodes by title or topic.

Explore themed categories like Health, Society, Education, and Culture.

The layout is fully responsive, optimized for desktops, tablets, and mobile devices.

Features

Responsive Design: Works across different screen sizes.

Audio Player: Global audio playback for all episodes.

Episode Grid: Organized main podcast grid and category-based sections.

Search Bar: Search for specific episodes with an icon inside the input.

Interactive Buttons: Play buttons and “See All Episodes” button with hover effects.

Footer Section: Includes links, social media icons, and legal information.

Technologies Used

HTML5 – Semantic markup for structure.

CSS3 – Custom styling, responsive layouts, flexbox, and grid.

JavaScript – For interactive audio playback and hamburger menu functionality (optional).

Audio Files – .m4a format used for podcast episodes.
